Process Control Overview

About

Process Control is an application for controlling your computer-activity. It provides user with abiblity to disable opening chosen executables and browsing chosen urls. Application allows user to create 'tasks' with set of forbidden urls and executables, add progress to them and finally start them for a while.

Used technologies

Application is build using some Dll-injection and hooking techniques. Here is the list of existing solutions, which were used in Process Control application somehow:

  • Minhook
  • Dll Injector

User interface made with Qt 5.4.0 Community edition.
