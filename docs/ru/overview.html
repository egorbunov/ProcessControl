Обзор программы Process Control

Общая информация

Process Control - это приложение для контролирования действий пользователя в за компьютером. Пользователь может выбрать набор программ и интернет-адресов, которые во время работы программы буду "запрещены". Так же программа позволяет создавать разные режимы работы, каждый из которых содержит свой набор запрещённых сайтов и программ, а так же свой счётчик прогресса :). Пользователь может включать тот или иной режим на любой промежуток времени, тем самым улучшая свою производительность труда или чего-нибудь ещё!

Используемые технологии

Приложение построено с использованием некоторых техник инъекции динамических библиотек (Dll) в сторонние процессы и перехвата вызовов функций. Ниже приведены готовые решения, которые были как-то использованы в программе:

  • Minhook
  • Dll Injector

Графический интерфейс сделан с помощью Qt 5.4.0 Community edition.
